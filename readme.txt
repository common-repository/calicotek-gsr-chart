== Description ==
This is a custom made plugin for london coin to update there gold silver ration stats. someone else may find it usefull but it was made primarily for a gold company.

== Installation ==
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. configure on the settings page to set the current spot prices.
4. Use this shortcode [cctgsrchart] in any page to show the chart output page to customers on front of site.
5. All Done :)
